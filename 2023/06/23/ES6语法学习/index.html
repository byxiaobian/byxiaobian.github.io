1. let and const

1. let

  1. let用于声明变量

  2. 有局部作用域

  3. let声明的变量不会提升

2. const

  1. const用于声明常量

  2. const声明的常量也不会提升

  3. const声明对象时,该对象不可以更换,但是对象里的内容可以更改

2. 箭头函数

1. 使用方法

  1. 理解为一般函数去掉function关键字再在参数和函数语句块中间加上=>

  2. 如果只有一个参数,可以去掉括号(注意没有参数时也要加上()占位)

  3. 如果只有一条返回语句,可以再去掉return和{ }

  4. 箭头函数在返回对象时必须在对象外层多加一个()

  5. this 与普通函数的this不同, 普通函数谁调用这个函数,this就是指向谁 箭头函数没有自己的this,它的this继承于父级, 也就是箭头函数的this指向箭头函数所处对象

3. 数组新增的高级方法

三个方法:

  1. filter()
  2. map()
  3. reduce()

他们都会接收一个函数,然后数组有多长,就执行多少次函数 相当于循环这个数组并每次对数组元素使用这个函数

5. 字符串新增方法与模板字符串

1. 字符串新增方法

1. startswith(string)

String.startswith(string) 判断String是否以string开头 返回true或false

2. endswith(string)

String.endswith(string) 判断String是否以string结尾

2. 模板字符串

即字符串的增强版

  1. 可以在其中随意书写, 所写即所得,不像普通的字符串一样换行需要特殊符号,换行书写需要多加引号
  1. 可以在字符串中引入其他变量,而不必像普通字符串一样写为 string + 变量 + string
